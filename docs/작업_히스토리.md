# 랜덤뽑기 작업 히스토리

## 프로젝트 개요
HTML5 기반 랜덤 추첨 애플리케이션 개발 (2026-03-18)

## 주요 기능 개발 순서

### 1. 기본 화면 구성 및 애니메이션
- **배경 및 레이아웃**
  - Background.png (581x423) 비율 유지
  - 반응형 레이아웃 (모바일/태블릿/PC 대응)
  - vw, vh, vmin 단위 사용

- **카드 애니메이션**
  - Card1~6.png 랜덤 선택
  - 3D 회전 애니메이션 (3회 회전)
  - 0.05배에서 1배로 확대
  - 2초 동안 연출
  - 카드 중앙에 뽑힌 이름 표시 (하늘색, 진한 파란색 그림자)
  - Card.mp3 사운드 재생
  - pick.mp3 사운드 (이름 표시 시)

- **Paper Unit 애니메이션**
  - paper_01.png ~ paper_10.png 순차 애니메이션
  - 텍스트 클리핑 효과 (상단 41% 위치)
  - 검붉은색 텍스트 (#8B0000)
  - 좌측 정렬, 3줄은 10개씩, 나머지 2줄은 5개씩 배치
  - 명단 개수만큼만 표시

- **별 효과**
  - Star.png 랜덤 생성
  - 상단 절반 영역에 랜덤 배치
  - 1~2배 크기 랜덤
  - 90도 회전하면서 페이드 인/아웃
  - 가중치 랜덤 간격: 0.5초(10%), 1초(20%), 1.5초(30%), 2초(30%), 2.5초(10%)

### 2. 인터랙션
- **토끼 버튼**
  - rabbit_button_normal.png, rabbit_button_over_push.png
  - 좌측 하단 10%, 크기 27%
  - 클릭 시 Click.mp3 재생
  - 0.5초 후 카드 애니메이션 시작
  - 애니메이션 진행 중 중복 클릭 방지

- **연출 순서**
  1. 토끼 버튼 클릭 → Click.mp3
  2. 0.5초 대기
  3. 카드 애니메이션 (2초) → Card.mp3
  4. 이름 표시 → pick.mp3
  5. 카드 애니메이션 종료 후 토끼 버튼 재활성화
  6. 0.5초 후 paper 애니메이션 시작 (독립 실행)

### 3. 명단 관리 시스템
- **인덱스 기반 추첨**
  - 중복 이름 허용 (인덱스로 구분)
  - 카드에 표시: "이름 (명단번호)" 형태
  - Paper에 표시: "이름"만 표시
  - 최대 40명 제한

- **localStorage 저장**
  - 브라우저에 명단 자동 저장
  - 기본 명단: 1~40

- **파일 관리**
  - 파일 불러오기: UTF-8, EUC-KR, CP949 자동 인식
  - 공유하기: Web Share API 사용
    - 안드로이드: 명단.txt 파일로 다양한 앱에 공유
    - PC: Fallback으로 다운로드

- **UI**
  - 반응형 모달 (80vw, 80vh, 최대 1.3 비율)
  - 줄 번호 표시
  - 텍스트/버튼 크기 vmin 단위로 조정

### 4. 반응형 디자인
- **컨테이너 비율 유지**
  - 581:423 비율
  - max-width: calc(100vh * 581 / 423)
  - max-height: calc(100vw * 423 / 581)

- **텍스트 크기 자동 조정**
  - 카드 텍스트: 18cqw (container query width)
  - Paper 텍스트: 25cqw
  - 줄바꿈 방지: white-space: nowrap

- **버튼 배치**
  - 컨트롤 버튼: 좌측 하단 2% (명단 관리, 리셋)
  - 전체화면 버튼: 우측 상단 1%, 크기 5%, z-index 150

### 5. 전체화면 및 모바일 최적화
- **전체화면 시작 안내**
  - 페이지 로드 시 오버레이 표시
  - "전체화면으로 시작" / "건너뛰기" 선택

- **가로 모드 강제**
  - Screen Orientation API 사용
  - 전체화면 진입 시 landscape로 자동 회전
  - iOS/Android 호환성 포함

- **파비콘**
  - 토끼 버튼 이미지를 32x32로 리사이즈
  - base64로 인코딩하여 HTML에 포함

### 6. 최적화 및 통합
- **이미지 최적화**
  - rabbit_button_over_push.png: 596KB → 85KB
  - 전체 파일 크기: 1.9MB → 515KB (73% 감소)

- **Standalone HTML 생성**
  - 모든 이미지/사운드를 base64로 인코딩
  - 동적 경로를 JavaScript 배열로 변환
  - 단일 HTML 파일로 통합 (randomChoice_standalone.html)
  - 최종 크기: 약 700KB

## 주요 변경 이력

### 카드 애니메이션
- 회전 횟수: 5회 → 4회 → 3회
- 시작 크기: 0.1 → 0.05
- 속도: 2초로 조정

### Paper 위치
- 클리핑 영역: 45% → 44% → 43% → 41%
- 텍스트 좌측 여백: 20px → 25px → 28px → 30px → 20% (비율)

### 토끼 버튼
- 크기: 15% → 30% → 25.5% → 27%
- 위치: bottom 5% → 10%

### Paper Unit 배치
- 초기: 10개씩 자동 배치
- 최종: 3줄(10개) + 2줄(5개) 고정 배치

### 명단 관리
- 초기: 이름 기반 중복 체크
- 최종: 인덱스 기반 중복 허용

### 파일 저장
- 초기: 다운로드만 지원
- 최종: Web Share API로 공유 기능 추가

  - Audio API
