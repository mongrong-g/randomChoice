# 랜덤뽑기 프로젝트 개요

## 프로젝트 설명
엽기토끼 캐릭터를 활용한 HTML5 기반 랜덤 추첨 애플리케이션입니다.
모바일/태블릿/PC 모든 환경에서 사용 가능하며, 단일 HTML 파일로 배포됩니다.

## 주요 기능

### 1. 랜덤 추첨
- 토끼 버튼 클릭으로 명단에서 랜덤 선택
- 카드 애니메이션으로 결과 표시
- 중복 추첨 방지 (인덱스 기반)
- 40명까지 지원

### 2. 애니메이션 연출
- **카드 연출**: 3D 회전 + 확대 (2초)
- **Paper 연출**: 10프레임 애니메이션 + 텍스트 클리핑
- **별 효과**: 랜덤 생성, 회전 페이드
- **사운드**: 클릭, 카드, 선택 효과음

### 3. 명단 관리
- 브라우저 자동 저장 (localStorage)
- 파일 불러오기 (UTF-8, EUC-KR, CP949 지원)
- 공유하기 (Web Share API)
- 줄 번호 표시

### 4. 모바일 최적화
- 전체화면 모드
- 가로 화면 강제
- 터치 최적화
- 반응형 레이아웃

## 파일 설명

### randomChoice.html
원본 파일. 이미지와 사운드를 외부 파일로 참조합니다.
- 개발/수정 작업에 사용
- 용량: 약 32KB

### randomChoice_standalone.html
배포용 통합 파일. 모든 리소스가 base64로 내장되어 있습니다.
- 단일 파일로 배포 가능
- 용량: 약 700KB
- 이 파일만 있으면 어디서든 실행 가능

## 사용 방법

### 초기 설정
1. `randomChoice_standalone.html` 파일을 브라우저로 열기
2. "전체화면으로 시작" 선택 (권장)
3. 모바일: 가로 모드로 자동 전환

### 명단 관리
1. 하단 "명단 관리" 버튼 클릭
2. 한 줄에 하나씩 이름 입력 (최대 40명)
3. "저장" 버튼 클릭
   - 브라우저에 자동 저장됨
   - 다음 방문 시에도 유지됨

### 명단 백업/공유
- **파일 불러오기**: .txt 또는 .csv 파일 선택
- **공유하기**:
  - 모바일: 다른 앱으로 공유 (드라이브, 메모, 카톡 등)
  - PC: 파일 다운로드

### 추첨 진행
1. 토끼 버튼 클릭
2. 카드 애니메이션 확인 (뽑힌 이름 표시)
3. Paper 위치에 결과 자동 표시
4. 40명 모두 추첨 시 안내 메시지

### 리셋
- "리셋" 버튼: 추첨 결과 초기화
- 명단은 유지됨

## 기술 상세

### 반응형 설계
- 기본 비율: 581:423 (Background 이미지 비율)
- 화면 크기에 따라 자동 조정
- 모든 요소가 비율에 맞춰 크기 조정

### 애니메이션 타이밍
```
토끼 클릭
    ↓ Click.mp3
   0.5초 대기
    ↓
카드 애니메이션 (2초)
    ↓ Card.mp3
이름 표시
    ↓ pick.mp3
토끼 버튼 활성화 (다음 추첨 가능)
    ↓ 0.5초 후
Paper 애니메이션 (1초)
```

### 중복 이름 처리
같은 이름이 여러 개 있어도 각각 다르게 취급됩니다.

**예시 명단:**
```
1
2
3
1
2
3
...
```

**카드 표시:**
- "1 (1)" - 1번째 위치의 1
- "1 (4)" - 4번째 위치의 1

**Paper 표시:**
- 모두 "1"로 표시 (위치 번호 없음)

### 데이터 저장
- **브라우저 localStorage**: 명단 자동 저장
- **키 이름**: `randomChoiceNameList`
- **추첨 상태**: 페이지 새로고침 시 초기화
